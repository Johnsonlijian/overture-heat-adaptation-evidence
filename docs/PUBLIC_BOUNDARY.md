# Public Repository Boundary

This repository contains reproducibility code, derived source-data tables, source registries and generated figures for audit and review.

It intentionally excludes active manuscripts, cover letters, confidential review correspondence, internal logs, raw third-party datasets, downloaded geometry caches, private author files and credentials. Raw Overture, GHSL, 3DBAG, GlobalBuildingAtlas and NYC Open Data inputs must be obtained from the original providers listed in `DATASETS_AND_LINKS.csv`.
